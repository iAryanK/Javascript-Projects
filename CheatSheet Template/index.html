C Plus Plus CheatSheet
  1. C++ Boiler plate
    This is the basic C++ boiler plate code. 
    #include <iostream>
using namespace std;

int main() {
cout << "Hello World";
return 0;
}
  2. cout <<
    This prints output on the terminal window 
    cout << "This is C++ Programming";
  3. cin>>
    This is to take user input. 
    cin << variable_name;
  4. Strings
    It is a collection of characters surrounded by double quotes. 
    // Include the string library
#include <string>

// String variable
string variable1 = "Hello World";
  5. If-else-if statement
    Conditional statements used to perform some operation based on some condition. 
    if (condition) {
// Statements;
}
else if (condition){
// Statements;
}
else{
// Statements
}
  6. Switch Case statement
    It allows a variable to be tested for equality against a list of values (cases). 
    switch (expression) 
{
case constant-expression: 
statement1;
statement2;
break;
case constant-expression: 
statement;
break;
...
default: 
statement;
};
